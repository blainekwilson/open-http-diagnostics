# Platform Conformance Guides

Platform guides explain how existing products satisfy the OHD specification. They do not redefine OHD fields or requirements.

Each platform may contain up to four level documents. A missing file means the level has not yet been documented.

Current guides:

- [IIS](iis/README.md)
- [NGINX](nginx/README.md)
- [Apache HTTP Server](apache/README.md)
- [Envoy](envoy/README.md)
- [HAProxy](haproxy/README.md)
